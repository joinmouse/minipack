## 📦 Minipack

现代JS模块打包器的简易实现、详细教学

本仓库基于原版[minipack](https://github.com/ronami/minipack)修改，改动如下：

- 升级了依赖的`babel`版本

- 全中文注释

- 增加了一些辅助调试的代码

如果你喜欢这个教学，请给[原作者仓库](https://github.com/ronami/minipack)一个`star`吧

## 使用

```shell
# 按照依赖
npm install;
# 运行打包脚本
npm run build
```

打开`src/minipack.js`学习
